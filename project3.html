Real-Time AI Based Traffic Management System

🚦Real-Time AI Based Traffic Management System

Author: Deepak Annapureddy

Tools Used: Python, OpenCV, YOLOv11, Flask

🔍 Project Overview

  • This project is an AI-based traffic management system that detects vehicles from CCTV/video feeds and dynamically controls traffic signals.
  • It uses YOLOv11 for real-time object detection and calculates traffic density based on vehicle count.

🎯 Project Objectives

  • Detect vehicles in real-time using deep learning
  • Estimate traffic density (Low / Medium / High)
  • Dynamically adjust traffic signal timing
  • Provide ambulance priority system
  • Display live dashboard using Flask

📊 Project Outputs

⚙️ System Working

  • Input video is captured from CCTV or file
  • Frames are processed using OpenCV
  • YOLOv11 detects vehicles in each frame
  • Vehicle count determines traffic density
  • Signal timing is adjusted dynamically

📈 Results

  • Achieved 95–97% detection accuracy
  • Real-time processing at 20–30 FPS
  • Efficient in dense traffic conditions
